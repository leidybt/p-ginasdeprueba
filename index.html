Nuestras Ideas

Al decidir una idea para nuestro proyecto de Laboratorio, primero presentamos ideas individuales por cada integrante del equipo, de esa forma pudimos discutir acerca de una idea general que englobe a las demás.

Componentes

Determinamos los componentes que usaremos para nuestro proyecto de Laboratorio. A continuación, se muestra una lista de los componentes electrónicos y sus principales funciones.

Diagrama de Flujo

Previamente a diseñar nuestro circuito, realizamos nuestro diagrama de flujo, el cual nos ayudó a determinar procesos. De esta manera, fue mucho más fácil programar en el Arduino.

Descarga nuestro diagrama de flujo AQUÍ

Armado de Circuito

Con todos los componentes adquiridos, procedimos al armado del circuito. Asimismo, fue necesario soldar algunos componentes para evitar que se salgan del protoboard.

Puedes ver una parte del circuito en Tinkercad AQUÍ

Programación

Para la programación del circuito se usó Arduino IDE. Adicionalmente, se usaron dos librerías para controlar al DFPlayer y al Aro Neopixel.

Descarga la librería del Neopixel AQUÍ

Planos de Prototipo

A continuación se muestran los planos del prototipo del proyecto. El diseño se basó en un pokemón llamado Gengar.

Diseño 3D

Para el modelado de nuestro Gengar, nos basamos en una geometría de Thingverse. Adicionalmente usamos los programas Blender y Tinkercad para adptarlo a nuestro proyecto.

Parámetros de Impresión

El programa laminador fue Cura. La impresora que se configuró fue una Ender 5 Pro. Adjuntamos los parámetros que usamos en Relleno, Soportes, Retracciones, etc.

Fabricación

El proceso de impresión 3D de nuestro Gengar fue largo, ya que se contaba con 4 partes o archivos para imprimir. También, el uso de huecos para los ojos y bocina resultó en una mayor cantidad de soportes.

Post-Procesado

Para un mejor acabado de la impresión 3D, se usó masilla para tapar las uniones. Igualmente, se usaron pinturas en spray para brindarle un color uniforme.

Implementación

Con el post-procesado finalizado, se procedió a incluir el circuito. Para ello, se soldó una batería de 9V para alimentar al Arduino.
